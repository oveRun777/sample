使用方法
新しいディレクトリを作成後、当該ディレクトリでコンソールを立ち上げ、

npm install -D

を実行する。コンソールで、

gulp

と入力して実行。
ローカルサーバが立ち上がり、監視開始。
あとは、working_dirに作業ファイルを入れて作業開始。
（gulpfile.babel.jsを編集した後は一度コンソールを止める。）

public_dir（ディレクトリ）
working_dir（ディレクトリ）
.babelrc
package.json
gulpfile.babel.js
